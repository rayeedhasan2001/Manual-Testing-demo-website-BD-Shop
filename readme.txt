BD Shop — Manual Test Case Writing Project

A manual SQA project focused on writing test cases for the Sign Up and Sign In module of the BD Shop e-commerce web application.


Project Overview
This project involves manually designing and documenting test cases for the User Management module of BD Shop, specifically the Sign Up and Sign In features. The goal is to ensure the functionality, usability, and reliability of these flows through structured manual testing.


Scope of Testing

Module ->Features Covered 
User Management ->Sign Up, Sign In


Testing Types Applied

Browser Compatibility Testing — verifying the site works across different browsers
UI Testing — fonts, alignment, mandatory field indicators, checkboxes, hover text
Functional Testing — field validations, CAPTCHA, Remember Me, Forgot Password, password rules
Usability Testing — keyboard tab/enter navigation, copy-paste behavior
Negative Testing — invalid inputs, blank fields, special characters, boundary values
